## 📌　アプリ概要
### アプリ名  
Bright Balance ～日記で整える、心と生活～
### 説明  
このアプリは日々の日記記録にプラス思考や自分では考えつかなかった視点を取り入れることを目的としました。また、自分のストレス度を把握することで積極的なストレスケアを促します。
### 背景
日々、忙しくて自分の状態を理解できない中で
### 主な機能  
- ✅ 日記内容をプラス思考やアドバイスを付加した形で変換
- ✅ ストレスチェックを行うことで日々のストレス度を可視化
### 今後の実装予定機能
- 変換内容のお気に入り登録
- ストレスチェック結果の一覧表示
